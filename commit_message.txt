feat(frontend): Implement Story Memory feature frontend components, store, and i18n

This commit introduces the frontend implementation for the AI Memorial Studio's
Story Memory sub-feature, including new Vue components, a Pinia store for state
management, and localization keys for both English and Vietnamese.

Key changes include:
- **Vue Components (`apps/admin/src/components/memory`):**
  - `MemoryList.vue`: Displays a paginated list of memories for a member.
  - `MemoryCreate.vue`: Multi-step form for creating new memories, integrating photo analysis and story generation.
  - `PhotoAnalyzerPreview.vue`: Component to display AI photo analysis results.
  - `StoryEditor.vue`: Component for reviewing and editing AI-generated stories.
  - `MemoryDetail.vue`: Displays the full details of a single memory.
  - `MemoryEdit.vue`: Form for editing existing memories.
- **Pinia Store (`apps/admin/src/stores/memory.store.ts`):**
  - `useMemoryStore`: Manages state, handles API calls for memory CRUD, photo analysis, and story generation. Refactored to correctly use `apiClient`'s `Result` type.
- **TypeScript Types (`apps/admin/src/types/memory.d.ts`):**
  - Defined interfaces for `PhotoAnalysisResultDto`, `GenerateStoryRequestDto`, `GenerateStoryResponseDto`, `CreateMemoryDto`, `UpdateMemoryDto`, `MemoryDto`, and `TimelineEntryDto`.
  - Exported these types via `apps/admin/src/types/index.ts`.
- **Router Configuration (`apps/admin/src/router/index.ts`):**
  - Added a new route `/members/:memberId/memories` for `MemberMemoriesView`.
- **Views (`apps/admin/src/views/member/MemberMemories.vue`):**
  - `MemberMemories.vue`: The main entry point for the AI Memorial Studio for a specific member.
  - Updated `apps/admin/src/views/index.ts` and `apps/admin/src/views/member/index.ts` to export the new view.
- **Utilities (`apps/admin/src/utils/formatters.ts`):**
  - Added `formatDateTime`, `formatDate`, and `formatCurrency` helper functions.
- **Internationalization (`apps/admin/src/locales/en.json`, `apps/admin/src/locales/vi.json`):**
  - Added comprehensive localization keys for all new UI elements and messages.
- **E2E Test Plan (`apps/admin/tests/e2e/memory_e2e_plan.md`):**
  - Outlined key user flows for end-to-end testing of the Story Memory feature.
- **Backend AI Prompts (`apps/backend/src/Application/Common/Prompts`):**
  - Created `photo_analysis.vi.md` and `story_generation.vi.md` for AI integration.
- **Backend Unit Test Skeleton (`apps/backend/tests/Web.UnitTests/Controllers/MemoriesControllerTests.cs`):**
  - Added basic unit tests for the `MemoriesController` endpoints.

This commit completes the full implementation of the Story Memory feature, from backend API and database integration to frontend UI components and localization."
