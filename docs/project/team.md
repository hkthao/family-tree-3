## 3. Trách nhiệm (updated after refactor)

Phân công trách nhiệm rõ ràng giúp đảm bảo mọi khía cạnh của dự án đều được bao quát và mỗi thành viên đều biết vai trò của mình:

*   **Full Stack Developer (Huynh Kim Thao)**: 
    *   Thiết kế, phát triển và duy trì cả Backend (ASP.NET Core) và Frontend (Vue.js).
    *   Quản lý cơ sở dữ liệu (MySQL, Entity Framework Core Migrations).
    *   Đảm bảo kiến trúc hệ thống tuân thủ Clean Architecture và các best practices.
    *   Viết Unit Tests, Integration Tests và Component Tests cho cả Backend và Frontend.
    *   Phối hợp với AI Assistant để đảm bảo chất lượng code và tài liệu.

*   **AI Assistant (AI Gemini)**: 
    *   Hỗ trợ duy trì tính nhất quán và cập nhật của tài liệu dự án.
    *   Hỗ trợ các nhiệm vụ phát triển phần mềm như sửa lỗi, thêm tính năng, refactor code (bao gồm refactor kiến trúc Clean Architecture, triển khai API tải lên tệp, và đổi tên `auth0UserId` thành `externalId`).
    *   Đảm bảo tuân thủ các quy ước và best practices của dự án.
    *   Cung cấp thông tin và phân tích kỹ thuật khi cần thiết.
